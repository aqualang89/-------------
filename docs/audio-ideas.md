# Идеи звука для интро (аквариум)

## Варианты реализации

### 1. Готовый MP3 + Web Audio
- Файл: `public/audio/intro-ambient.mp3`
- Запуск: при первом движении мыши (autoplay blocked)
- Стоп: при закрытии интро
- Громкость: 0.4

### 2. Интерактивный звук (по движению мыши)
- Мышь двигается = шум воды (volume по скорости)
- Мышь стоп = тишина
- Реализация: Web Audio API (Brown noise + lowpass filter)

### 3. Гибрид
- Тихий фон ambient постоянно
- При движении мыши: splash/плески поверх
- Мышь стоп: только тихий фон
